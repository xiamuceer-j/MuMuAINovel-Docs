# 项目简介

MuMuAINovel 是一个基于 AI 的智能小说创作助手，旨在帮助创作者高效地构建世界观、管理角色、生成大纲，并辅助完成章节的撰写与润色。

## 核心特性

- 🤖 **多 AI 模型支持** - 集成 OpenAI、Gemini、Claude 等主流语言模型
- 📝 **智能创作向导** - AI 自动生成小说大纲、角色设定和世界观
- 👥 **可视化管理** - 人物关系图谱和组织架构的可视化管理工具
- 📖 **高级编辑功能** - 章节创建、编辑、AI 重新生成及智能润色
- 🔐 **灵活登录系统** - 支持本地账户和 LinuxDO OAuth 两种方式
- 💾 **企业级数据库** - PostgreSQL 确保数据稳定性和多用户隔离
- 🐳 **一键部署** - Docker Compose 配置，开箱即用

## 技术栈

**后端**: FastAPI • PostgreSQL • SQLAlchemy • OpenAI/Claude/Gemini SDK

**前端**: React 18 • TypeScript • Ant Design • Zustand • Vite
