# 配置 AI API（重要！）

::: warning 前置步骤
在开始创作之前，必须先配置 AI API 并测试连接，这是使用系统的前提条件。
:::

登录成功后，**首先需要配置 AI 模型**，否则无法使用 AI 生成功能。

## 进入设置页面

1. 点击页面右上角的用户头像或设置图标
2. 选择"设置"或"AI 配置"选项

![API 配置](/images/usage/api.png)

## 配置 API 信息

在设置页面中，你需要填写以下信息：

#### 1. 选择 AI 提供商

系统支持OpenAl Compatible格式API：
- **OpenAl Compatible**：GPT、Claude、Gemini、DeepSeek、Qwen 等模型

#### 2. 填写 API 配置

根据选择的提供商，填写对应信息：

**OpenAI 配置示例**：
- **API Key**：`sk-your-api-key-here`
- **Base URL**：`https://api.openai.com/v1`（使用官方 API）
- **Base URL**：`https://api.new-api.com/v1`（使用中转服务）

**其他配置项**：
- **模型名称**：如 `gpt-4o-mini`、`gpt-4`、`claude-3-sonnet` 等
- **Temperature**：控制生成的随机性（0-1），默认 0.7
- **Max Tokens**：单次生成的最大字数，建议 8000-32000

## 获取可用模型列表

配置好 API 信息后：

1. 点击"获取模型列表"按钮
2. 系统会自动从 API 提供商获取你可用的所有模型
3. 从下拉列表中选择你想使用的模型

![API 模型列表](/images/usage/api-model.png)

## 测试 API 连接

::: tip 重要提示
在正式使用前，强烈建议先测试 API 连接！
:::

1. 点击"测试连接"或"测试 API"按钮
2. 系统会发送一个测试请求到 AI 服务
3. 查看测试结果：
   - ✅ **成功**：显示测试响应内容，可以开始使用
   - ❌ **失败**：检查 API Key 是否正确、网络连接是否正常、余额是否充足

![API 测试](/images/usage/api-test.png)

## 常见 API 配置问题

**问题 1：API Key 无效**
- 检查是否正确复制了完整的 API Key
- 确认 API Key 是否已激活且有余额
- 某些服务可能需要等待几分钟才能生效

**问题 2：连接超时**
- 检查网络连接
- 如果在国内，可能需要使用代理或中转服务
- 尝试更换 Base URL 为国内可访问的中转服务

**问题 3：模型不可用**
- 确认你的账户有权限使用该模型
- 某些高级模型（如 GPT-4）需要特殊申请
- 可以先使用基础模型（如 gpt-3.5-turbo）测试

## 保存配置

配置并测试成功后：
1. 点击"保存"按钮
2. 系统会将配置保存到数据库
3. 后续创建项目时将自动使用这些配置
