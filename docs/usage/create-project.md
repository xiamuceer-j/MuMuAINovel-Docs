# 创建项目

配置好 API 后，进入项目列表页面，可以开始创建你的第一个项目。

![项目列表](/images/usage/project-list.png)

系统提供两种创建模式：

## 模式一：向导创建（推荐新手）

向导创建是传统的表单填写方式，适合已经有明确创作思路的用户。

#### 操作步骤

1. **点击"使用向导创建"按钮**

   ![向导创建-1](/images/usage/wizard.png)

2. **填写项目基本信息**：

   ![向导创建-2](/images/usage/wizard-2.png)
   - **书名**：你的小说标题
   - **小说简介**：用一段话介绍你的小说（最多300字）
   - **主题**：描述小说的核心主题（最多500字）
   - **类型**：选择或输入类型标签（如：玄幻、都市、修仙，可多选）
   - **叙事视角**：选择第一人称、第三人称或全知视角
   - **角色数量**：AI 自动生成的角色数量（3-20个）
   - **目标字数**：整部小说的目标字数（最少10000字）

3. **选择大纲章节模式**（重要！创建后不可更改）：

   **📋 传统模式 (1→1)**
   - 一个大纲对应一个章节
   - 简单直接，结构清晰
   - 适合：简单剧情、快速创作、短篇小说

   **📚 细化模式 (1→N) - 推荐**
   - 一个大纲可以展开为多个章节
   - 灵活控制，便于细化情节
   - 适合：复杂剧情、长篇创作、需要详细展开的故事

4. **开始创建**

   ![向导创建-3](/images/usage/wizard-3.png)

   点击"开始创建项目"按钮后，系统会自动进行三个步骤：
   
   - ✅ **生成世界观**：AI 根据你的设定生成时代背景、地理位置、氛围和规则
   - ✅ **生成角色**：自动创建指定数量的角色，包含姓名、性格、背景等完整信息
   - ✅ **生成大纲**：创建3个大纲节点（之后可在项目内手动展开为章节）

5. **等待完成**

   整个过程需要2-5分钟，请耐心等待。生成过程中可以看到实时进度。

::: tip 智能重试
如果某个步骤生成失败，系统会提供"智能重试"按钮，自动从失败的步骤继续，无需重新开始。
:::

## 模式二：灵感模式（推荐探索）

灵感模式是创新的对话式创作方式，通过与 AI 对话逐步确定项目设定，适合还在构思阶段的用户。

#### 操作步骤

1. **点击"灵感模式"按钮**

2. **开始对话**

   ![灵感模式-1](/images/usage/inspiration.png)

   AI 会问："你想写一本什么样的小说？"
   
   你可以自由描述你的想法，例如：
   - "我想写一本关于时间旅行的科幻小说"
   - "一个少年在异世界成长的冒险故事"
   - "现代都市中的修仙者"

3. **交互式确定设定**

   ![灵感模式-2](/images/usage/inspiration-2.png)

   AI 会根据你的想法，逐步引导你完成以下设定：
   
   - **书名**：AI 提供3个书名建议，你可以选择或自己输入
   - **简介**：AI 生成3个版本的简介供选择
   - **主题**：AI 提炼3个主题方向
   - **类型**：AI 推荐相关类型标签（可多选）
   - **视角**：选择叙事视角
   - **大纲模式**：选择一对一或一对多模式

4. **确认创建**

   ![灵感模式-3](/images/usage/inspiration-3.png)

   所有设定完成后，系统会展示完整摘要：
   ```
   📖 书名：xxx
   📝 简介：xxx
   🎯 主题：xxx
   🏷️ 类型：xxx
   👁️ 视角：xxx
   📋 大纲模式：xxx
   ```
   
   确认无误后，点击"✅ 确认创建"。

5. **自动生成**

   确认后，系统会像向导模式一样，自动生成世界观、角色和大纲。

#### 灵感模式的优势

- 💬 **对话式体验**：更自然、更有创意的创作过程
- 🎨 **AI 建议**：获得多个创意方向供选择
- 🔄 **灵活调整**：随时可以选择"重新开始"
- 💾 **自动保存**：对话进度自动保存，24小时内可恢复

::: warning 注意
灵感模式的对话进度会自动保存24小时。如果24小时内再次进入，会自动恢复上次的对话。
:::

## 两种模式对比

| 特性 | 向导创建 | 灵感模式 |
|------|---------|---------|
| **适合人群** | 有明确创作思路 | 还在构思探索 |
| **操作方式** | 填写表单 | AI 对话 |
| **创意支持** | 需要自己想好 | AI 提供建议 |
| **速度** | 快速直接 | 需要交互 |
| **灵活性** | 一次性填写 | 逐步调整 |
| **生成内容** | 相同 | 相同 |

选择哪种模式取决于你的个人偏好和创作阶段！
